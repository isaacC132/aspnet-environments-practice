# launchSettings.json {#launchsettingsjson  data-source-line="272"}

Define perfiles de ejecución locales para:

- `dotnet run`
- Visual Studio / VS Code

Cada perfil puede tener:

- `applicationUrl`
- `launchBrowser`
- `environmentVariables` → `ASPNETCORE_ENVIRONMENT`
